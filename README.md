# angularWebpackLumx

## Purpose
Acting as boilerplate of angular + ui router + webpack + lumx for Hongtoo Frontend Team 

## Quick start

Two Steps Only:

- Clone the repo: git clone https://github.com/fancn21th/angularWebpackLumx.git.
- Install with NPM: `npm install`.
